== Overview ==
Allow users to set a custom primary menu for each page. This plugin adds meta boxes to the post edit screen where users can select a custom menu for the header and footer.

== Installation ==
1. Install via the WordPress Plugin Manager or
2. Upload the plugin files to the `/wp-content/plugins/` directory

== Usage ==
1. On the post edit screen, find the "Custom Menu Selector" meta boxes and select the desired menus.
2. After saving the post, the selected menus will be displayed in the corresponding positions in the theme.
